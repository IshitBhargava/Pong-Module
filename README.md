# Pong-Module
This is simple module using the main CyberPong game!

# Things needed:
<pre>
   Python IDLE (or any other IDE)
   Python Package Manager (pip)
</pre>

# Installing Packages:

<pre>
   before running it, you must install the following libraries:
   Pygame
   Sounddevice
   Textwrap
   Os
   Numpy
</pre>
